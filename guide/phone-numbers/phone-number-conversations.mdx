Whenever a user sends a message, this page will automatically update to reflect the new conversation.
A new conversation is created between the assistant's phone number and the user's phone number, ensuring that all interactions are promptly displayed and managed in real time.

## Conversations Route

The Conversations route provides an interface to view, search, and manage WhatsApp conversations associated with a specific phone number.

## Main Features

- **Conversation List:**
  Displays all conversations for the selected phone number. You can search by contact name or message content, and filter conversations using tags or assignment status.

- **Phone Number Profile:**
  Shows details about the phone number, including display name, verified name, and profile picture. You can update the profile picture and access settings dialogs.

- **Conversation Management:**
  Click on a conversation to view its details. You can export conversations in CSV or XLSX format for further analysis.

- **Settings and Customization:**
  Access dialogs to update phone number settings, display name, and profile information.

This route is designed to help users efficiently manage customer interactions, keep track of ongoing conversations, and customize phone number profiles for better organization and branding.
